Titikkita
=========

Kebutuhan Sistem
---------------------------------------------------------------------------------------
Terdapat beberapa kebutuhan software yang digunakan dalam pengembangan aplikasi Titikita diantaranya:

1. Android Studio > 4.2.0
2. Flutter SDK
3. Dart Plugins
4. Android Virtual Device / Emulator
